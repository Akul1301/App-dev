Place your images here, e.g.:

- profile.png        -> your profile picture (used on the Home screen)
- project1.png ...    -> project screenshots (used on Project Detail screen)

Then update the corresponding `imageAsset` paths in
`lib/models/portfolio_data.dart` and uncomment the `Image.asset(...)` lines
in `home_screen.dart` / `project_detail_screen.dart`.
